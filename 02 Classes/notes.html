Classes and Objects

What is a class?
  • blueprint for creating objects
  • one public class per file
  • any number of package private (no explicit modifier) classes per file
What is an object?
  • attributes
  • methods
  • create objects using new
Controlling Access to Members of a Class
  • public - full access for all
  • protected - access is restricted to derived classes and any classes in the same package
  • private - access restricted to class
  • package-private (no explicit modifier) - access restricted the the class and any classes in the same package
Constructors and finalizers
  • CTORs are for initializtion
  • DTORs are not useful
