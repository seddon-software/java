Lambdas

What is a Lambda?
  • inline functions
  • syntax suger
  • already available in many other languages: C++, C#, Python, Scala ...
How do you define a Lambda?
  • must give it a type: Functional Interface
  • must give it a body: (params) -> { body }
How do you use a Lambda?
  • pass the Lambda as a parameter
  • works with instance and static methods
