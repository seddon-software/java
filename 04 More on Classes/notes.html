More on Classes

clone method
  • defined in Object class to perform a shallow copy
  • doesn't call a constructor
  • must be called in a try block - awkward to use
copy Constructor
  • write your own constructor to copy objects
  • can be used instead of clone
  • often used in conjunction with clone
equals and hashCode
  • equals defined in Object class
  • override in your own class ...
  • ... but must use correct signature
  • ... and must also overload hashCode
